# 面试评分标准

## 评分维度与权重

| 维度 | 权重 | 评分范围 | 核心考察点 |
|------|------|---------|-----------|
| 内容完整度 | 30% | 0-100 | 是否覆盖问题的核心要点，回答是否有深度 |
| 逻辑条理性 | 25% | 0-100 | 表达是否清晰、层次分明、因果关系明确 |
| STAR 结构性 | 25% | 0-100 | 是否包含场景(S)、任务(T)、行动(A)、结果(R) |
| 关键词覆盖 | 20% | 0-100 | 是否使用了岗位相关的专业术语和关键词 |

## 分数段标准

### 内容完整度 (30%)

| 分数段 | 描述 |
|--------|------|
| 90-100 | 全面覆盖问题要点，补充了面试官未明问但相关的重要信息 |
| 75-89 | 覆盖了主要要点，有适当的细节支撑 |
| 60-74 | 覆盖了部分要点，但遗漏了重要的方面 |
| 40-59 | 仅覆盖少量要点，回答过于简略 |
| 0-39 | 几乎未涉及问题的核心内容 |

### 逻辑条理性 (25%)

| 分数段 | 描述 |
|--------|------|
| 90-100 | 表达清晰流畅，因果链完整，层次递进 |
| 75-89 | 整体有条理，个别地方跳跃但可理解 |
| 60-74 | 有基本结构，但逻辑跳跃较多 |
| 40-59 | 表达混乱，缺乏组织 |
| 0-39 | 完全没有逻辑结构 |

### STAR 结构性 (25%)

| 分数段 | 描述 |
|--------|------|
| 90-100 | 完整的 STAR 四要素，每个要素都有具体内容 |
| 75-89 | 包含 3 个以上 STAR 要素，R(结果) 有量化数据 |
| 60-74 | 包含 2 个 STAR 要素，缺少结果量化 |
| 40-59 | 仅包含 1 个 STAR 要素，缺乏结构 |
| 0-39 | 完全没有 STAR 结构，只是零散描述 |

### 关键词覆盖 (20%)

| 分数段 | 描述 |
|--------|------|
| 90-100 | 自然地使用了 5+ 个岗位相关专业术语 |
| 75-89 | 使用了 3-4 个专业术语，语境恰当 |
| 60-74 | 使用了 1-2 个专业术语 |
| 40-59 | 使用了非专业表述，但意思正确 |
| 0-39 | 表述含糊，缺乏专业术语 |

## 总分计算

```
总分 = 完整度 × 0.30 + 条理性 × 0.25 + STAR × 0.25 + 关键词 × 0.20
```

## Tier 分级

| Tier | 分数 | 含义 |
|------|------|------|
| S | >= 90 | 优秀：结构完整、内容深入、表达专业 |
| A | 75-89 | 良好：结构较完整、覆盖主要要点 |
| B | 60-74 | 合格：基本覆盖要点，有改进空间 |
| C | 40-59 | 需改进：结构不完整，关键信息缺失 |
| D | < 40 | 不合格：缺乏基本结构和内容 |

## STAR 完整性检查清单

- [ ] **Situation**: 是否描述了具体的背景/场景？
- [ ] **Task**: 是否明确了自己的职责/目标？
- [ ] **Action**: 是否用强动词描述了具体行动？（如：主导、策划、优化）
- [ ] **Result**: 是否包含量化的成果？（如：提升30%、节省2万、覆盖500人）
- [ ] **连贯性**: S→T→A→R 之间是否有逻辑递进？
- [ ] **相关性**: 经历是否与目标岗位的能力要求相关？
